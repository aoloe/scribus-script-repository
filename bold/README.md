# Setting the font face

This script can be used to test ideas on how to apply the bold face for _every_ font.

You're welcome to contribute heuristic that can help getting there.

If the result is convincing, we will try to redo the logic as Scribus code and it as a feature and allow setting bold (and italic!) with keyboard shortcuts.

With Scribus 1.7 you can already use this script to apply bold with the short cut:

ctrl+/ bo
